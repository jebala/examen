Este producto que se solicita consiste en la creación de una calculadora que mediante el trabajo con números naturales los sumara, restara, multiplicara, calculara el error de la resta y trabajara con la notación de números romanos. Por lo tanto las historias de usuario que lo compondrán sera:

1) Suma, esta historia usuario sumara dos números naturales

2) Multiplicar, en esta historia de usuario se realizara la multiplicacion de dos numeros naturales.

3) Resta, esta historia de usuario consistira en restar dos números naturales y como aqui se solicita que el valor del resultado sea natural, sino lo fuera daria lugar a la siguiente historia de usuario.

4) Error de resta, sería cuando nose pudiera realizar la suma no daria un número natural daría uno negativo

5) Números romanos, en esta historia de usuario se utilizara la notación de estos números para ayudar a los niños a aprenderlos.

SPRINT

1 Sprint--> En este primer Sprint nos compremeteremos a entregar las dos primeras historias de usuario.
	- Suma, le daremos una duración de una semana.
	- Multiplicar, le daremos la duración de una semana

2 Sprint--> En este segundo Sprint nos compremeteremos a entregar las historias de usuario restantes.
	- Resta, esta historia de usuario se le estima el mismo valor de una semana
	- Error de resta, esta historia de usuario de unos 4 días
	- Números romanos, se le estima una duración de una semana

La priorización del cliente la dará el orden anteriormente descrito desde la primera historia de usuario y la más prioritaria hasta la ultima y menos prioritaria que incluso es opcional que es la de números romanos.
